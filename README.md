# 🦇 BATMAN: GOTHAM DILEMMA
**A terminal-based detective adventure built with JavaScript.**

## 🏙️ The Story
The rain in Gotham doesn't just fall; it punishes. The Bat-Signal has been corrupted by a jagged, neon-green question mark. The Riddler has teamed up with Scarecrow to poison the city's water supply. You are Batman. You are alone. Every choice you make determines whether Gotham breathes easy or falls into a slow, agonizing madness.

## 🧠 Key Features
Detective Logic: Solve riddles involving time, reflections, and shadows to uncover the true path.

Branching Narrative: Multiple paths including the Mirror Grand Hotel, Gotham Waterworks, and the Clock Tower.

Multiple Endings: Your choices lead to a Perfect Victory, a Partial Success, or a Failure that leaves Gotham at risk.

## 🛠️ Tech Stack
Language: JavaScript (Node.js)

Architecture: State-based "Story Node" logic.

Version Control: Git & GitHub

## 📜 Project Structure
`script.js`: Contains the game engine and the four core functions (getCurrentScene, displayScene, makeChoice, and restartGame).

`storyNodes`: A comprehensive object mapping the narrative flow and choice-based ID routing.

## How to Run

1. Clone the repository to your local machine.
2. Ensure Node.js is installed.
3. Run the following command in your terminal: node script.js
4. FOR QUICK AND EASY ACCESS, ONCE NODE IS INSTALLED. TYPE "node " and drag the folder itself into the terminal, then press ENTER.
